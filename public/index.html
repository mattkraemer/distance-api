Distance Calculator API

Introduction

The Distance Calculator API provides a simple method to calculate the geographical distance between two sets of latitude and longitude coordinates. It uses the Haversine formula to ensure accurate results.

Endpoint

GET http://localhost:3000/:lat1-:lng1/:lat2-:lng2

Parameters

  • lat1, lng1: Latitude and Longitude of the first location.
  • lat2, lng2: Latitude and Longitude of the second location.
  • unit (optional): Desired unit for the result distance. Can be 'kilometers' (default), 'meters', or 'miles'.

Response

Returns a JSON object with the calculated distance and the unit of measurement.

        {
          "distance": 1234.56,
          "unit": "kilometers" // Can be "kilometers", "meters", or "miles"
        }

If no unit is specified in the request, the default unit will be "kilometers".

Example

To calculate the distance between New York City and Los Angeles in miles:

http://localhost:3000/40.7128,-74.0060/34.0522,-118.2437/miles
